Dzung's weather app
- A non-profitable, made with free and open-source tools Weather Web Application

BUILDING FLOW:
	1.	Project + Webpack setup
	2.	Basic HTML scaffold
	3.	API fetch logic (async/await)
	4.	JSON processing
	5.	Form submission handling
	6.	DOM rendering
	7.	Polish (loading state, dynamic imports, styling)

    •	api.js → fetch data
	•	dom.js → update UI
	•	index.js → handle events


FEATURES:
- Global weather coverage
- Supports English, (updating...) 
- Low latency
- Automatic location detection
- Responsive layout
- Support light and dark theme
